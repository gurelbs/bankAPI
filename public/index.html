Wellcome to BankAPI

nice and simple API, Created by Gurel Ben Shabat Just to learn how API realy work!

cheatsheet:

  • bankAPI.cf
  • POST
    bankAPI.cf/api/create?q=[number]
  • GET
    bankAPI.cf/api/users
  • GET
    bankAPI.cf/api/users?id=[id]
  • PUT
    bankAPI.cf/api/users/?id=[id]&cash=[number]&credit=[number]
  • PUT
    bankAPI.cf/api/users/[id]/withdraw?amount=[number]
  • PUT
    bankAPI.cf/api/users/[id]/withdrawP2P?amount=[number]&to=[id]

    • Create:

      POST
    • /api/create?q=[number] (positive integer number between 1 - 100) 
      • exmple: https://bankapi.cf/api/create?q=100
      • exmple: http://localhost:3000/api/create?=13

    Show:

      GET
    • all users: /api/users
    • user by id: /api/users?id=[id]
      • exmple: https://bankapi.cf/api/users?id=iQGbAk72oTr6KkuUwYeU9E
      exmple: http://localhost:3000/api/users?id=iQGbAk72oTr6KkuUwYeU9E

    Mange User Cash & Credit:

      PUT
    • deposit cash for a user: /api/users?id=[id]&cash=[number]
    • update user credit: /api/users?id=[id]?credit=[number]
    • update user credit & cash: /api/users?id=[id]&cash=[number]&credit=[number]
      • exmple: http://localhost:3000/api/users?id=iQGbAk72oTr6KkuUwYeU9E&cash=5000
      exmple: http://localhost:3000/api/users?id=iQGbAk72oTr6KkuUwYeU9E&credit=1000
      exmple: http://bankapi.cf/api/users?id=iQGbAk72oTr6KkuUwYeU9E&credit=1000&cash=5000

    Mange User Withdraw:

      PUT
    • withdraw cash from a user: /api/users/[id]/withdraw?amount=[number]
      • exmple: https://bankapi.cf/api/users/gww9jL6S64TBAz1SvPtiRX/withdraw?amount=1500

    Mange Users P2P Transaction:

      PUT
    • withdraw cash from user to user: bankAPI.cf/api/users/[id]/withdrawP2P?amount=[number]&to=[id]
      • exmple: bankAPI.cf/api/users/iQGbAk72oTr6KkuUwYeU9E/withdrawP2P?amount=1300&to=gww9jL6S64TBAz1SvPtiRX
