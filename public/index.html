Wellcome to BankAPI

nice and simple API, Created by Gurel Ben Shabat Just to learn how API realy work!

Cheatcheet:

  • bankAPI.cf
  • bankAPI.cf/api/create/[number]
  • bankAPI.cf/api/users
  • bankAPI.cf/api/users/[id]
  • bankAPI.cf/api/users/[id]?cash=[number]&credit=[number]
  • bankAPI.cf/api/users/[id]/withdraw?amount=[number]
  • bankAPI.cf/api/users/[id]/P2Pwithdraw?amount=[number]?to=[id]

    • Create:

      GET
    • /api/create/[number] (positive integer number between 1 - 100) 
      • exmple: https://bankapi.cf/api/create/5
      • exmple: http://localhost:3000/api/create/5

    Show:

      GET
    • all users: /api/users
    • user by id: /api/users/[id]
      • exmple: https://bankapi.cf/users/gww9jL6S64TBAz1SvPtiRX
      exmple: http://localhost:3000/users/gww9jL6S64TBAz1SvPtiRX

    Mange User Cash & Credit:

      PUT
    • deposit cash for a user: /api/users/[id]?cash=[number]
    • update user credit: /api/users/[id]?credit=[number]
    • update user credit & cash: /api/users/[id]?cash=[number]&credit=[number]
      • exmple: http://localhost:3000/users/gww9jL6S64TBAz1SvPtiRX?cash=900
      exmple: https://bankapi.cf/users/gww9jL6S64TBAz1SvPtiRX?credit=16500
      exmple: https://bankapi.cf/users/gww9jL6S64TBAz1SvPtiRX?cash=1500&credit=2000

    Mange User Withdraw:

      PUT
    • withdraw cash from a user: /api/users/[id]/withdraw?amount=[number]
      • exmple: https://bankapi.cf/users/gww9jL6S64TBAz1SvPtiRX/withdraw?amount=1500

    Mange Users Transaction:

      PUT
    • withdraw cash from a user: /api/users/[id]/P2Pwithdraw?amount=[number]?to=[ID]
      • exmple: https://bankapi.cf/users/gww9jL6S64TBAz1SvPtiRX/P2Pwithdraw?amount=1500?to=g9e6FWULxKUEpNV3fPyR6e
